What is Hexabones ?

Hexabones wants to be the solid foundation to start a brand new e-commerce design. It's made to work close with PrestaShop in order to provide the best user experience, where guideline meets human. This meant that we want to provide to designers and developpers a complete and time saving tool for creating amazing and efficient themes for PrestaShop.

What it isn't ?

  • A production Theme
  • An offial PrestaShop project

Want to be part ?

Hexabones is an open-source project, you can participate to its conception by discussing major decisions on the community boards or pulling some great lines of code on the git repo.
